## about

opentelemetry 是基础库代码其他仓库，然后在 `otelcol、otelcol-contrib、otelcol-k8s` 在 [opentelemetry-collector-releases](https://github.com/open-telemetry/opentelemetry-collector-releases) 内使用 [ocb](https://github.com/open-telemetry/opentelemetry-collector/tree/main/cmd/builder) 读取 [manifest.yaml](https://github.com/open-telemetry/opentelemetry-collector-releases/blob/v0.101.0/distributions/otelcol-contrib/manifest.yaml) 生成项目结构去构建的。
